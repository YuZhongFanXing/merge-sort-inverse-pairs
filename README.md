# 归并排序逆序对计数动画演示

一个交互式动画演示，展示如何使用**归并排序**高效地计算数组中的**逆序对**。通过可视化和动画，帮助学习者深入理解这个经典算法问题。

## 🎯 功能特性

- **交互式数组可视化**：实时显示排序过程中的数组状态
- **分步动画演示**：支持播放、暂停、前进、后退和速度调整
- **算法状态追踪**：显示当前逆序对数量、已排序范围、合并范围等
- **原理讲解**：包含详细的算法原理、时间复杂度分析和应用场景
- **数据验证**：使用暴力法验证算法结果的正确性
- **自定义输入**：支持输入自定义数组或随机生成数组

## 📊 什么是逆序对？

在数组中，如果 `i < j` 但 `arr[i] > arr[j]`，则称 `(arr[i], arr[j])` 为一个**逆序对**。

**例子：** 数组 `[3, 1, 2]` 有两个逆序对：`(3, 1)` 和 `(3, 2)`

## 💡 算法原理

### 时间复杂度

- **暴力方法**：O(n²) - 检查所有元素对
- **归并排序方法**：O(n log n) - 利用分治思想

### 核心洞察

在合并两个已排序的子数组时，当**左子数组的元素大于右子数组的元素**时，左子数组中从该位置到末尾的所有元素都与右子数组的元素形成逆序对。

### 算法步骤

1. **分割**：递归地将数组分成两半
2. **排序**：对每个子数组递归排序
3. **合并**：合并两个已排序的子数组
4. **计数**：在合并时计算逆序对数量

## 🎨 可视化说明

- **深青色**：已排序的元素
- **橙色**：当前正在合并的元素
- **浅黄色**：正在分割的元素
- **红色**：检测到的逆序对
